About

This site was built by Samuel Jackson (slj11) for Assignment 2 of CS15020. All pages in this site are valid to XHTML 1.1 and CSS 2 W3C standards. I approached this project by designing a basic html template for a single page, styling it to look nice. and then copying the XHTML structure to all the child pages and link them with a single stylesheet.

I found this task relatively easy to complete, having plenty of previous experianc ein XHTML and CSS development back in college, however I did run into one small snag in attempting to make the footer "stick" to the bottom of the page, no matter how large the content area is. To solve this I used a small work around by Ryan Fait.

This site was developed using Notepad++ and Firefox Nightly builds. It has also been tested in all of the latest stable releases of popular web browsers, including Chrome, Firefox and IE.

The Javascript used in this page uses the traditional event registration model. I wanted to use the more up recent W3C event registration model, however, due to backwards compatibility issues with IE, this was not an option for this assignment. The traditional event registration model is standard in nearly all browsers and removes the use of inline Javascript; thus seperating the stucture from functionality.
